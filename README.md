# WarriorsOfTheForest
By: Łukasz Jagiełło, Wiktor Górecki 4PTB


A simple game made in Java Swing.

## Players:
You can choose two classes:
- **Warrior**
- **Archer**
- **Healer**




## Enemies:
There is only one enemy - monster
